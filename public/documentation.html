API Documentation

Movies Endpoints

Request URL Description Response
GET / This sends back the message “Welcome to my movie club!” as text in the body of your HTTP response. "Welcome to my movie list!"
GET /movies Get a list of all movies Returns an array of movie objects
GET /movies/:title Returns a movie when the user searches by title Returns movie title
GET /movies/:id Get details of a specific movie Movie object with the specified ID
PUT /movies/:id Update details of a specific movie The updated movie object
DELETE /movies/:id Delete a specific movie Confirmation message
